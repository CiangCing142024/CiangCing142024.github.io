आईएल अंतर्राष्ट्रीय लाल बचाव , संक्षिप्त नाम के साथ भी जाना जाता है मोप मूल संप्रदाय से (में) रशो क्रांति सेनानियों को सहायता का अंतर्राष्ट्रीय संगठन ? , " क्रांति के सेनानियों के लिए अंतर्राष्ट्रीय लाल बचाव "), यह एक अंतरराष्ट्रीय संगठन था अंतर्राष्ट्रीय कम्युनिस्ट स्थापना करा 1922 संगठन ने कैदियों का समर्थन करने के लिए सामाजिक एकजुटता के कुछ अभियानों का नेतृत्व किया कम्युनिस्टों और विशेष स्थितियों में सामग्री और मानवीय समर्थन। इसका नेतृत्व किया गया था क्लारा ज़ेटकिन , ऐलेना स्टासोवा इ टीना मोडोटी ।

अंतर्राष्ट्रीय लाल बचाव 18 मार्च, 1923 को पेरिस के नगरपालिका की सालगिरह पर घोषित किया गया, राजनीतिक कैदियों के साथ अंतर्राष्ट्रीय एकजुटता दिवस '

राष्ट्रीय उत्सर्जन की सूची

अल 1924 , संगठन में 19 राज्यों में राष्ट्रीय उत्सर्जन था। उनमें से उन्होंने गिना: [१] [५] [[] [[] [९]

  • ऑस्ट्रिया : ऑस्ट्रियाई लाल बचाव ( ऑस्ट्रियाई लाल मदद )
  • ग्रेट ब्रिटेन : वर्ग संघर्ष के कैदियों के लिए अंतर्राष्ट्रीय बचाव
  • बुल्गारिया : पूंजीवादी तानाशाही के पीड़ितों का समर्थन करने के लिए संगठन
  • फ्रांस : अंतर्राष्ट्रीय लाल बचाव ( अंतर्राष्ट्रीय लाल सेकोर्स )
  • जर्मेनिया : जर्मनी का लाल बचाव ( जर्मनी में लाल मदद )
  • इटली : इतालवी लाल बचाव
  • मेक्सिको : सताए गए सेनानियों के लिए समर्थन के लिए लीग ( समर्थक सेनानियों का पीछा किया )
  • नीदरलैंड : नीदरलैंड के बचाव लाल ( रूड मदद हॉलैंड )
  • नॉर्वे : नॉर्वे का लाल बचाव ( नॉर्वे की रोड मदद )
  • पोलैंड : राजनीतिक कैदियों के लिए समर्थन आयोग
  • सोवियत संघ : क्रांति के सेनानियों के लिए अंतर्राष्ट्रीय लाल बचाव ( क्रांति सेनानियों को सहायता का अंतर्राष्ट्रीय संगठन )
  • स्पेन : अंतर्राष्ट्रीय लाल बचाव ( अंतर्राष्ट्रीय लाल समाज )
  • स्वीडन : अंतर्राष्ट्रीय लाल बचाव - स्वीडिश अनुभाग ( इंटरनेशनल रेड एड - स्वीडिश सेक्शन )
  • यूएसए : श्रम की अंतर्राष्ट्रीय रक्षा ( अंतर्राष्ट्रीय श्रम रक्षा )
